This map covers all four GK Gold retail services — Scrap/Cash for Gold, Bullion Buying & Selling, Gold Loan/Pawnbroking (AMS), and Storage. Eight AI agents are proposed to automate the highest-friction, highest-risk areas: KYC intake, live pricing, transaction processing, inventory management, AML/CTF integration, loan lifecycle, storage management, and daily reconciliation reporting. The system connects Xero (retail), CashNet (loans), and Win Gold (storage) under a unified automation layer.

Compliance Threshold Reference — Know Before Every Transaction
$5,000
CDD Trigger — Bullion Only
Buying or selling any bullion (gold bars, coins, silver bars) for $5,000 or more triggers CDD/KYC under s39E AML/CTF Act. Applies to walk-in and online sales.
Unique to bullion dealers. Most assume $10,000 — it is $5,000.
$10,000
CDD Trigger — Scrap/Cash for Gold
Standard AML/CTF CDD threshold for scrap gold/silver transactions. Cash threshold only — EFT does not trigger. Also: $10,000 cash = TTR obligation for ALL services.
All GK Gold designated services — TTR within 10 business days.
$0
SMR — No Threshold
Suspicious Matter Reports have NO minimum dollar amount. Any suspicious behaviour, regardless of transaction value, must be escalated to the AML/CTF Compliance Officer.
Report suspicion, not amount. Tipping-off is a criminal offence.
$5,000 = CDD for bullion  ·  $10,000 = CDD for scrap  ·  $10,000 cash = TTR for all services  ·  No threshold for SMR  ·  2% cash surcharge applies across all services

AI Agent Architecture — 8 Agents for Retail Operations

Agent 1 — KYC Intake Agent
Triggers GBG greenID on customer intake. Collects data from POS/website, submits for verification, receives result, writes outcome and risk rating to compliance register automatically.

Agent 2 — Live Pricing Agent
Pulls live spot price via API every 60 seconds. Auto-calculates buy/sell prices, loan values, and pool allocation pricing. Eliminates manual spot price checking and CashNet silent failures.
Feeds → Agent 3 (Xero), Agent 6 (Loans)

Agent 3 — Transaction Agent
Auto-creates Xero invoices and purchase orders on transaction confirmation. Records payment method, applies SKU codes, manages the quote-to-invoice lifecycle for online orders.
Feeds → Agent 5 (AML flags), Agent 8 (Reports)
Box
Agent 4 — Inventory Agent
Tracks crude/fresh stock levels across all branches. Flags when crud stock is available for resale. Monitors vault contents against inventory register. Alerts on low stock or discrepancies.
Feeds → Agent 3 (Xero), Agent 8 (Reports)
Shield
Agent 5 — AML/CTF Co-Pilot
Monitors all transactions for threshold triggers ($5K/$10K). Flags TTR obligations. Detects structuring patterns. Escalates to Compliance Officer. Interfaces directly with compliance Agent 2/3/4.
Feeds → Compliance SOPs (TTR/SMR), Agent 8
Lock
Agent 6 — Loan Lifecycle Agent
Manages CashNet loan status transitions (Active → Default → Expired → Acquired). Sends automated SMS at all reminder milestones. Auto-applies storage fees on default. Triggers acquisition workflow at Day 97.
Feeds → Agent 5 (AML), Agent 8 (Reports)
Vault
Agent 7 — Storage Agent
Manages Win Gold storage records for both allocated and pool allocation. Generates Bailor-Bailee agreements, triggers e-sign workflow, manages recurring billing, produces FY holdings reports and monthly statements.
Feeds → Agent 1 (KYC), Agent 8 (Reports)
Chart
Agent 8 — Reporting Agent
Auto-generates daily reconciliation reports (cash, EFT, Xero vs NAB). Produces transaction registers per lot/branch. Generates management dashboard. Triggers end-of-day sign-off workflow.
Feeds → Directors dashboard, Compliance CO

Agent 6 — Loan Lifecycle Auto-Management

Status Auto-Transitions

Active (Day 1-90) → Default (interest unpaid 30+ days, storage fees auto-apply) → Expired (Day 90, 7-day grace) → Acquired (Day 97). Each transition auto-triggers notifications to management.

Agent 6
SMS Reminders (Automated)

5 days before interest due · 3 days before · 2 days after unpaid · Expiry warning · Final notice before acquisition (Day 88 — formal written notice generated). No manual triggers.

Agent 6 · SMS
Storage Fees Auto-Applied

On DEFAULT status trigger: storage fees auto-apply (1%/day or $10/day, whichever higher — subject to Regulation max). Auto-waived when interest paid on time at 90-day mark.

Agent 6
Acquisition Workflow

Day 97: system triggers acquisition checklist. Auto-generates AMS-to-GK intercompany invoice at live scrap rate. Surplus tracked for 12-month trust period with auto-alert at public trustee transfer date.

Implementation Roadmap

  • Set up manual EOD reconciliation template as bridge (Agent 8 pre-build)
  • Engage lawyer: confirm interest rate caps and storage fee maximums
  • Draft AMS-GK intercompany contract
Phase 2 — Weeks 3-6
  • Build Agent 2: live spot price API integration (Xero + CashNet + Win Gold)
  • Build Agent 3: Xero transaction automation (invoices, POs, payment matching)
  • Build Agent 5: AML/CTF threshold monitoring + CO dashboard v1
  • Build Agent 6 (Phase 1): loan lifecycle status automation + SMS cadence
  • Auto-reference number generation (Branch + sequential PO)
  • Verify Win Gold e-sign for Bailor-Bailee agreements
Phase 3 — Weeks 7-12
  • Build Agent 1: GBG greenID API full integration into POS + website checkout
  • Build Agent 4: inventory management (crud/fresh, vault vs register)
  • Build Agent 7: Win Gold storage automation (billing, agreements, pool conversion)
  • Build Agent 8: EOD reconciliation + daily reporting automation
  • Agent 6 Phase 2: acquisition workflow, intercompany invoice, trust account tracking
  • Custom staff interface (replace full Xero/CashNet access with limited UI)
Target State — Post Week 12
  • KYC: fully automated via GBG greenID + ongoing screening
  • Pricing: live API — zero manual spot checking
  • Transactions: Xero auto-invoiced, NAB payment auto-matched
  • Loans: CashNet lifecycle fully managed by Agent 6
  • Storage: Win Gold fully automated by Agent 7
  • EOD: full reconciliation pack ready in 15 minutes via Agent 8
  • AML/CTF: Agent 5 monitors all services in real time
